## 🚀 Featured Project

### [Food-Waste-Management (Food Donation)](https://github.com/katiyarabhay/Food-Waste-Management)
🌐 [Live Demo](https://food-waste-management-topaz.vercel.app)

A web application dedicated to reducing food waste by connecting donors with those in need. The platform facilitates seamless donations of surplus food, promoting sustainability and community welfare.

**Main Features:**
- Easily post or find available food donations in your area
- Real-time listing of food supplies and requests
- Simple, user-friendly interface for both donors and recipients
- Promotes food sustainability and waste reduction
